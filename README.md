🛒 Manyam Mart POS System

A cloud-based Point of Sale (POS) system built using Flutter Web and Firebase for small retail stores.
This system enables real-time billing, inventory management, customer tracking, and sales analytics.

---

🚀 Live Demo

👉 https://web-app-5bccf.web.app

---

✨ Features

- 🧾 Real-time billing system
- 📦 Product & inventory management
- 📊 Sales dashboard with daily analytics
- 👥 Customer details storage (name & mobile)
- 🔍 Barcode scanning support
- 🖨 Thermal receipt generation & printing
- ⚠ Low stock alerts
- 🔐 Admin login system
- 🌐 Cloud-based (accessible from anywhere)
- 📱 Responsive (works on desktop & mobile)

---

🛠 Tech Stack

- Frontend: Flutter Web
- Backend: Firebase
- Database: Cloud Firestore
- Authentication: Firebase Auth
- Hosting: Firebase Hosting

---

📌 How It Works

1. Admin logs into the system
2. Products are added via admin panel
3. Cashier adds products to cart (manual/barcode)
4. Customer details are entered
5. Bill is generated
6. Receipt is printed
7. Sales data is updated in dashboard
